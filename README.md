## Mini Project: Find Available Hotels on Booking
This small project combines **Selenium** and **PySide6** to create a desktop application with a graphical interface that automatically retrieves a list of available hotels for today from Booking.com in the selected city.
**Features:**
- Automated hotel availability scraping
- Simple GUI built with PySide6
- Modular structure (main.py, parser.py, gui.py)

## Мини-проект: поиск свободных отелей на Booking
Этот небольшой проект обьединяет **Selenium** и **PySide6** для создания приложения с графическим интерфейсом, которое автоматически собирает список свободных отелей на сегодня с сайта Booking.com в выбранном городе.
**Особенности:**
- Автоматический сбор данных об отелях
- Простой графический интерфейс
- Модульная структура (main.py, parser.py, gui.py)
